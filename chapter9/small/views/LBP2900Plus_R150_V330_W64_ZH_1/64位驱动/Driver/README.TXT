1. 缩放

  如果指定缩放，某些应用程序可能会打印不正确。在此情况下，请将缩
  放比例设置为100%。
  如果应用程序有其自己的缩放功能，请从应用程序中设置缩放。

================================================================

2. 页面布局

  某些应用程序提供打印多份副本的功能。不过，如果在 [页面设置] 选
  项卡 [页面布局] 选择了 [每张 1 页] 以外的其它设置，则不要使用
  该功能。
  如果在应用程序中使用该功能，则数据可能排列不正确。

================================================================

3. 分隔页

  在 Windows 中无法打印 [分隔页]。

================================================================

4. 安装时的用户权限
   （Windows 2000/XP/Vista/7/Server 2003/Server 2008）

  以 Administrators 成员用户的身份登录以安装打印机驱动程序。要进
  行安装，您必须具有完全控制权限。

================================================================

5. 在安装后使用 Microsoft Office

  如果您正在使用安装程序，并且有提示信息要求您重新启动计算机而您
  没有重新启动，随后 Microsoft Office 可能会操作不正确。如果发生
  这种情况，请重新启动计算机。

================================================================

6. 卸载其他用户连接到的网络打印机
   （Windows 2000/XP/Vista/7/Server 2003/Server 2008）

  如果您在计算机中执行 [Canon LBP2900Plus 卸载程序]，而其他用户
  已连接到 Canon LBP2900Plus 网络打印机服务器，则无法删除其他已
  连接用户的 Canon LBP2900Plus 网络打印机的信息。在此情况下，其
  他用户应首先删除其 Canon LBP2900Plus 打印机图标，之后，您才能
  执行卸载程序。

================================================================

7. 对于网络打印机的限制
   （Windows 2000/XP/Vista/7/Server 2003/Server 2008）

  如果在 Windows 2000/XP/Vista/7/Server 2003/Server 2008 中安装
  了网络打印机，即使将其连接目的地更改为本地端口，该打印机还是无
  法用作本地打印机。
  如果将该打印机用作本地打印机，而不是已安装的网络打印机，请选择
  本地端口，然后再次安装打印机驱动程序。

================================================================

8. 有剩余打印作业时重新启动 Windows
   （Windows 2000/XP/Vista/7/Server 2003/Server 2008）

  如果需要在有剩余 Canon LBP2900Plus 打印作业时重新启动 
  Windows，则从打印机文件夹中暂停打印作业，然后重新启动 
  Windows。

================================================================

9. 下载和安装网络打印机
   （Windows 2000/XP/Vista/7/Server 2003/Server 2008）

  在 Windows 2000/XP/Vista/7/Server 2003/Server 2008 中下载和安
  装网络打印机时，您需要以具有管理员权限的用户身份来安装它。
  如果网络打印机是由没有管理员权限的用户安装，则由于操作系统安全
  方面的限制，某些功能可能会无法使用。

  [无法使用的功能]
  - 打印机状态窗口的声音功能不能正确工作。
  - 卸载程序无法注册到 [添加/删除程序] 中。
    （Windows XP/Server 2003 为 [添加或删除程序]；Windows 
    Vista/7/Server 2008 R2 为 [卸载程序]；Windows Server 2008 为
    [程序和功能]）

================================================================

10. 在多用户环境中安装（Windows XP/Vista/7/Server 2008）

  在 Windows XP/Vista/7/Server 2008 中安装 Canon LBP2900Plus 
  时，不要为通过 Windows XP/Vista/7/Server 2008 的“快速用户切
  换”功能切换的用户进行安装。
  在使用“快速用户切换”功能时，确保安装之前再次登录。

================================================================

11. Internet Explorer 的保护模式（Windows Vista/7/Server 2008）

  如果使用的是 Windows Vista/7/Server 2008 且通过单击光盘安装程
  序中的 [软件说明] 等图标运行 Internet Explorer，无论设置如何，
  [保护模式] 将设置为 [禁用]。
  如果要将 [保护模式] 设置为 [启用]，请退出 Internet Explorer，
  然后再次手动运行浏览器。

================================================================

12. 在多用户环境中显示打印机状态窗口
    （Windows XP/Vista/7/Server 2008）

  如果通过 Windows XP/Vista/7/Server 2008 的“快速用户切换”功能
  更改了用户，则在打印机状态窗口中可能无法显示打印机状态。要显示
  状态，请再次登录，然后打开打印机状态窗口。

================================================================

13. CAPT Windows 防火墙实用程序

  如果您使用配备了 Windows 防火墙的操作系统（如 Windows XP 
  Service Pack 2）作为打印服务器，则其与客户机之间的通信受 
  Windows 防火墙控制。[CAPT Windows 防火墙实用程序] 是一种配置工
  具，用于将 Windows 防火墙配置为允许或禁止与客户机之间的此类通
  信。
  [CAPT Windows 防火墙实用程序] 位于提供的 [用户软件] CD-ROM 上
  的 [Simpchin] - [WF_UTIL] 文件夹中。
  有关 [CAPT Windows 防火墙实用程序] 的详细信息，请参阅《用户指
  南》中的第 7 章“Windows 防火墙”。

================================================================

14. 关于使用打印机共享功能的客户端设置

  如果您使用的是 Windows XP Service Pack 2 或另一种在客户端配备
  了 Windows 防火墙的操作系统，请配置以下设置。如果您不在客户端
  指定以下设置，打印机状态窗口中显示的打印机状态可能会不正确，而
  且您可能无法使用打印机的某些功能。

  （如果在指定以下设置时出现 [用户帐户控制] 对话框，请单击 
  [继续] 或 [是]。）

  - 对于 Windows XP
    (1) 从 [开始] 菜单中选择 [控制面板]，然后单击[网络和 
        Internet 连接] -> [Windows 防火墙]。
    (2) 在 [Windows 防火墙] 对话框中的 [例外] 页上，选中 [文件和
        打印机共享] 复选框，然后单击 [确定]。

  - 对于 Windows Vista
    (1) 从 [开始] 菜单中选择 [控制面板]，然后单击 [允许程序通过 
        Windows 防火墙]。
    (2) 在 [Windows 防火墙设置] 对话框中的 [例外] 页上，选中 [文
        件和打印机共享] 复选框，然后单击 [确定]。

  - 对于 Windows 7
    (1) 从 [开始] 菜单中选择 [控制面板]，然后单击 [系统和安全] 
        -> [允许程序通过 Windows 防火墙]。
    (2) 单击 [更改设置]，选中 [文件和打印机共享] 复选框，然后单
        击 [确定]。

  - 对于 Windows Server 2003
    (1) 从 [开始] 菜单中，选择 [控制面板] -> [Windows 防火墙]。
    (2) 在 [Windows 防火墙] 对话框中的 [例外] 页上，选中 [文件和
        打印机共享] 复选框，然后单击 [确定]。

  - 对于 Windows Server 2008
    (1) 从 [开始] 菜单选择 [控制面板]，然后双击 [Windows 防火
        墙]。
    (2) 单击 [Windows 防火墙] 对话框中的 [更改设置]。
    (3) 在 [Windows 防火墙设置] 对话框中的 [例外] 页上，选中 [文
        件和打印机共享] 复选框，然后单击 [确定]。

  - 对于 Windows Server 2008 R2
    (1) 从 [开始] 菜单中选择 [控制面板]，然后单击 [检查防火墙状
        态] -> [允许程序或功能通过 Windows 防火墙]。
    (2) 单击 [允许的程序] 对话框中的 [更改设置]。
    (3) 选中 [文件和打印机共享] 复选框，然后单击 [确定]。

15. 打印机名称字符（Windows Vista/7/Server 2008）

  打印机名称中使用了“@”字符时，操作系统将因为打印机名称无效而
  识别不了打印机，从而导致不能打印、操作和设置。

================================================================

16. 注册自定义纸张尺寸（Windows Vista/7/Server 2008）

  确保在编辑打印机驱动程序中 [页面设置] 选项卡的 [自定义纸张尺
  寸] 时，以具有管理员权限的用户身份登录。

================================================================

17. 打印机属性的访问权限（Windows Vista/7/Server 2008）

  当以具有管理员权限的用户身份登录时，如果与安装打印机驱动程序时
  使用的用户身份不同，就不能访问打印机属性。
  要访问打印机属性，请执行下列两项操作之一。

  - Windows Vista 和 Server 2008
    右键单击打印机图标，然后选择 [用管理员帐户运行]。

  - Windows 7 和 Server 2008 R2
    单击打印机属性 [常规] 选项卡中的 [更改属性]。

================================================================

18. 打印细线（Windows Vista/7/Server 2008）

  在某些应用程序中，可以将线宽设置为小于 1 pt，但实际打印输出的
  线宽可能相当于 1 pt 线宽的线条。

================================================================

19. 从 Windows Vista 升级到 Windows 7 时的注意事项

  将操作系统从 Windows Vista 升级到 Windows 7 时，首先卸载打印机
  驱动程序，然后升级操作系统，升级后重新安装打印机驱动程序。
  如果在未卸载打印机驱动程序的情况下升级操作系统，打印机驱动程序
  可能无法正常工作。
  在这种情况下，先卸载打印机驱动程序，然后重新安装打印机驱动程
  序。
